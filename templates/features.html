Features

Server Wide Sharing

Server invites are shared among all the servers with the bot.

Automatic Entries

Entires are added to the database automatically when the bot joins a server, providing a seamless experience to the user.

Server Specific Command Prefix

Bot prefixes can be changed using the setprefix command.

Channel Customization

Default sharing channel can be customized using the setchannel command.

Server Descriptions

Server descriptions can be changed using the description command.
